# Support

We are dedicated to providing support for users and developers interacting with this portfolio repository.

## How to Get Help

If you have a question, issue, or need help with this project, please follow these steps:

1. **Check the Documentation:** Review the `README.md` and `CONTRIBUTING.md` for existing guidelines and project setup instructions.
2. **Search Existing Issues:** Before reporting a new problem, please search [our GitHub Issues](https://github.com/ermradulsharma/mradulsharma/issues) to see if someone else has already reported it.
3. **Open a New Issue:** If you cannot find an answer or a similar reported bug, please open a new issue. Use the provided Issue Templates in the repository to ensure you include all necessary details (e.g., bug description, expected behavior, environment info).

## Security Vulnerabilities

If you discover a security vulnerability within this project, please do **NOT** open a public issue. Instead, refer to our [`SECURITY.md`](SECURITY.md) guidelines for instructions on how to securely disclose the information to the maintainers.

## Community

Feel free to reach out for collaborations and professional inquiries directly through the contact forms available on the live portfolio site.
